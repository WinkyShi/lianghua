# 量化交易 - 中芯国际行情分析

中芯国际（688981.SH）近一年日线行情可视化面板。

## 内容

- **K线面板** ([在线浏览](https://winkyshi.github.io/lianghua/)) — 交互式 K线图 + 成交量图
- 数据来源：东方财富 API（前复权日线）
- 数据区间：2025-07-04 ~ 2026-07-03，共 236 个交易日

## 功能

- 日线 K线图（阳线红 / 阴线绿，A股惯例）
- 均线：MA5、MA10、MA20、MA60
- 成交量柱状图（涨红跌绿，与K线联动）
- 交互：缩放、拖拽、十字光标、hover 详情
- 顶部统计卡片：最高价、最低价、日均量、最大量、总成交额

## 文件说明

| 文件 | 说明 |
|------|------|
| `中芯国际_K线面板.html` | 可视化面板（GitHub Pages 在线浏览） |
| `中芯国际_688981_日线行情.csv` | CSV 格式行情数据 |
| `中芯国际_688981_日线行情.json` | JSON 格式行情数据 |
| `fetch_smic_data.py` | 数据获取脚本 |
| `generate_html.py` | HTML 面板生成脚本 |

## 技术栈

- 数据获取：Python + 东方财富 API
- 可视化：ECharts 5
